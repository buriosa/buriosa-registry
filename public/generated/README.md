- **registry.json** - Main component registry (Object format: `{ [name]: RegistryEntry }`)
- **page-registry.json** - Page components only
- **category-index.json** - Category → component IDs mapping
- **tag-index.json** - Tag indices (functional/style/layout/industry)
- **page-index.json** - Page → section IDs mapping
- **section-to-page.json** - Section → page reverse lookup
